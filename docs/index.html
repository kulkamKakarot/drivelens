A closer look

From drive picker to freed-up gigabytes in a few clicks.

Built to be trusted

Deleting files needs care. DriveLens is engineered around safety.

  • Recycle Bin only — the app has no code path for permanent deletion. Every removal is recoverable.
  • Confirmation with full detail — you see the exact paths and total size before anything moves.
  • System-path warnings — anything inside Windows or Program Files triggers an extra, explicit warning.
  • Sandboxed & validated — the UI runs in Chromium's sandbox; every file path is validated before the backend touches it.
  • Open code — ~1,500 lines of readable JavaScript, no obfuscation, auditable by anyone.
